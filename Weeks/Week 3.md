## Week Narrative

- **D1:** The knowledge cutoff problem — AI is stuck in the past. Tools are how it reaches the present.
- **D2:** What tools actually are — Interface vs API, MCP as the universal adapter standard.
- **D3–D5:** Three demos showing tool calling in practice — developer (Cursor), enterprise (customer support), creative (Remotion skill).
- **D6:** What happens when tool access goes wrong — 4 real, documented incidents with real consequences.
- **D7:** Marc Andreessen + Truth Terminal — $50K Bitcoin to an autonomous AI agent. This is where tool calling leads. Preview of Week 8.
